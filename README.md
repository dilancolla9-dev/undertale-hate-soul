# Undertale: Hate Soul 🎮

Un juego fan estilo Undertale donde juegas como el **Alma de Odio (Rosa)**.

## 🎨 Características Principales

### Protagonista: Hate Soul
- **Apariencia**: Pelo fucsia, camisa a rayas negro/rosa
- **Acompañante**: Fantasma rosado-rojizo que te sigue
- **Alma especial**: Representa el Odio

### Antagonista: Flowey Carnívoro
- Versión más oscura y hostil de Flowey
- Comportamiento de planta carnívora
- Diálogos únicos y malévolo
- Raíces que atacan

### Historia Única
- Trama completamente diferente a Undertale
- Personajes reinterpretados con aspectos más oscuros
- Nuevos personajes exclusivos
- Zonas rediseñadas

## 📁 Estructura del Proyecto

```
undertale-hate-soul/
├── project.godot              # Configuración de Godot
├── scenes/
│   ├── main.tscn            # Escena principal
│   ├── ui/                   # Interfaz del juego
│   └── zones/
│       ├── zone_1_fall.tscn  # LA CAÍDA
│       └── zone_2_flowey.tscn # JARDÍN OSCURO
├── scripts/
│   ├── player.gd            # Lógica del jugador
│   ├── flowey.gd            # Lógica de Flowey
│   ├── ghost.gd             # Fantasma compañero
│   └── dialogue_system.gd   # Sistema de diálogos
├── assets/
│   ├── sprites/              # Sprites pixel art
│   ├── tilesets/             # Tilesets de zonas
│   └── music/                # Música y sonidos
└── data/
    └── dialogues.json        # Base de diálogos
```

## 🚀 Cómo Ejecutar

1. **Descargar Godot 4.x**: https://godotengine.org
2. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/dilancolla9-dev/undertale-hate-soul.git
   ```
3. **Abrir en Godot**: Selecciona la carpeta del proyecto
4. **Ejecutar**: F5 o botón Play

## 📝 Notas de Desarrollo

- Motor: **Godot Engine 4.x**
- Lenguaje: **GDScript**
- Resolución base: **640x480** (estilo retro)
- Tamaño de sprites: **32x32 píxeles**
- Estilo: **Pixel Art** (inspirado en Undertale)
